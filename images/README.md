# Images Folder

This folder is for storing animal photography images for the website.

## Instructions for Adding Images

### Recommended Image Specifications:

- **Format**: JPG or PNG
- **Resolution**: Minimum 1920x1080px for best quality
- **File Size**: Optimize images to 200-500KB for web performance
- **Aspect Ratio**: 16:9 or 4:3 work best with the current layout

### How to Add Your Images:

1. **Prepare Your Images**:
   - Edit and optimize your animal photos using image editing software
   - Rename files with descriptive names (e.g., `lion-portrait.jpg`, `bird-in-flight.jpg`)
   - Ensure images are properly compressed for web use

2. **Upload to This Folder**:
   - Click "Add file" → "Upload files" in the GitHub repository
   - Drag and drop your images into this `images/` folder
   - Or use Git commands:
     ```bash
     git add images/your-image-name.jpg
     git commit -m "Add new animal photos"
     git push
     ```

3. **Update the HTML**:
   - Open `index.html`
   - Replace the placeholder image divs with actual image tags:
     ```html
     <img src="images/your-image-name.jpg" alt="Description of the image">
     ```

### Suggested Image Categories:

- **Wildlife**: Animals in their natural habitat
- **Pets**: Domestic animals and companion pets
- **Birds**: Various bird species and flight shots
- **Marine Life**: Underwater and aquatic animals
- **Close-ups**: Detailed shots of animal features
- **Action Shots**: Animals in motion

### Image Naming Convention:

Use descriptive, lowercase names with hyphens:
- ✅ Good: `african-elephant-sunset.jpg`
- ✅ Good: `golden-retriever-playing.jpg`
- ❌ Avoid: `IMG_1234.jpg`
- ❌ Avoid: `Photo 2023.jpg`

### Copyright Notice:

Ensure you have the rights to all images uploaded to this folder. Only upload:
- Your own original photographs
- Images you have licensed or purchased
- Royalty-free stock images with proper attribution

### Tips for Best Results:

1. Use high-quality, sharp images
2. Ensure good lighting and composition
3. Include variety in your portfolio (different animals, settings, and styles)
4. Consider the emotional impact of each image
5. Test images on both desktop and mobile devices
